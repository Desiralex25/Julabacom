⚠️ Pour activer l'envoi de SMS en production :

1. CONFIGURER LES SECRETS SUPABASE (3 minutes)
   
   Dans Supabase Dashboard → Edge Functions → Settings → Secrets :
   
   WASSOYA_API_KEY      = [Votre clé API Wassoya.com]
   WASSOYA_API_URL      = https://api.wassoya.com/sms/messages
   WASSOYA_SENDER_ID    = JULABA

2. REDÉPLOYER LES EDGE FUNCTIONS (1 minute)
   
   Terminal :
   $ supabase functions deploy make-server-488793d3

3. TESTER L'ENVOI DE SMS (1 minute)
   
   Terminal :
   $ ./test-wassoya-integration.sh 0701020304
   
   Ou sur : https://julabacom.vercel.app/
