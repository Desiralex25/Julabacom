## 🚨 URLs d'Urgence

### **Si tu es bloqué :**

1. **Déconnexion forcée :**
   ```javascript
   // Dans la console du navigateur :
   localStorage.clear();
   location.href = '/login';
   ```

2. **Réinitialiser le cache :**
   ```
   https://julabacom.vercel.app/?clear_cache=true
   ```

3. **Accès direct à une page :**
   ```
   https://julabacom.vercel.app/[role]/[page]
   ```
