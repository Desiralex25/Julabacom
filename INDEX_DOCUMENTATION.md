### **Comment créer le premier Super Admin ?**
📄 Voir : [`ACCES_RAPIDE_BACKOFFICE.txt`](/ACCES_RAPIDE_BACKOFFICE.txt)

### **Quelle est l'URL de connexion au Back-Office ?**
```
https://julabacom.vercel.app/backoffice/login
```
📄 Voir : [`URLS_REFERENCE.md`](/URLS_REFERENCE.md)

### **Quels sont les rôles disponibles ?**
- Super Admin
- Admin National
- Gestionnaire de Zone
- Analyste

📄 Voir : [`ARCHITECTURE_AUTH.md`](/ARCHITECTURE_AUTH.md) - Section "RBAC"

### **Comment sont gérés les tokens ?**
- Access Token : 1 heure
- Refresh Token : 7 jours
- Stockage : localStorage

📄 Voir : [`ARCHITECTURE_AUTH.md`](/ARCHITECTURE_AUTH.md) - Section "Gestion des Tokens"

### **Que faire si le compte est bloqué ?**
- Attendre 15 minutes
- Ou utiliser "Mot de passe oublié"

📄 Voir : [`README_BACKOFFICE_ACCESS.md`](/README_BACKOFFICE_ACCESS.md) - Section "Support"
